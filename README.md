# NIFTY Stock Market Prediction

This is a streamlit-based web application that predicts NIFTY stock prices using:
-ARIMA Model
-LSTM Model
-Hybrid Forecasting Approach

## Tech Stack
-Python
-Streamlit
-Statsmodel
-Tensorflow
-Scikit-learn
-Pandas
-NumPy

## How to run:
```bash
pip install -r requirements.txt
streamlit run app.py
